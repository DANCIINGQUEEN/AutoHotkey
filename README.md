# 텐키리스 키보드를 편히 쓰기 위한 AutoHotKey 스크립트 작성

# MC4.ahk
마우스 커서 상하좌우 직선운동 가속도 추가
ctrl+shift+alt+i : 상
ctrl+shift+alt+j : 좌
ctrl+shift+alt+k : 하
ctrl+shift+alt+l : 우
ctrl+shift+alt+u : 좌클릭
ctrl+shift+alt+o : 우클릭

ctrl+shift+alt+l : 우
